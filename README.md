# Sentinel - Autonomous IoT Security Platform

## Quick Start
```bash
npm install
cp .env.example .env
npm run dev
```

## Deploy to Vercel
```bash
npx vercel
# Set VITE_ANTHROPIC_API_KEY in Vercel dashboard
```

## Build
```bash
npm run build   # vite build (no tsc step)
```

## Tech Stack
React 18 + TypeScript + Three.js + Zustand + Vite + Claude API
